# Frontend Happimobi

Projeto Angular 18 inicial para consumir a API de reservas.

## Requisitos

- Node.js 20+
- npm 10+

## Comandos úteis

```bash
npm install
npm start
npm run build
```

## Docker

```bash
# build da imagem
docker compose build frontend

# subir somente o front
docker compose up -d frontend
```

A aplicação será exposta em `http://localhost:4200`.
